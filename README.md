Project Overview
This project is a website created by PK Electrical and Plumbing Services to allow customers a chance to find out more about us and the services we provide. This website allows customers to see our services, what we provide, and our history, and also lets them know more about us and how trustworthy we are with every task we are given. We value long-term trust, friendship, and meaningful partnerships, so we created an enquiry webpage that allows us to communicate with potential customers on whatever questions they have. This also includes our contact page for better communication.

Website Goals and Objective 
Goals
- To become the primary online destination for homeowners seeking reliable electrical and plumbing services
- To convert website visitors into qualified service leads through a seamless user experience  

Objective 
- Ensure 100% mobile responsiveness so customers can easily contact us from any device
- To ensure the website is fully responsive and accessible on mobile, tablet, and desktop devices.

Key Features
- Fully responsive layout using CSS Flexbox/Grid.
- Contact form for user submissions and enquiries.
- Consistent navigation and header across all pages.
